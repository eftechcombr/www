## Project Overview

The EF-TECH website (`www.eftech.com.br`) is the official marketing and content site for EF-TECH, a Brazilian company specialising in cloud computing solutions. Built as a Hugo static site with the Blowfish theme, it showcases services (cloud migration, security, DevOps, managed infrastructure), publishes a blog, and provides contact information for prospective clients. The site targets Portuguese-speaking (pt-br) audiences and is deployed automatically via GitOps to Cloudflare Pages and GitHub Pages.

## Technology Stack Summary

The site is a pure front-end static site generated by **Hugo** (Go-based SSG). No server-side runtime, database, or API backend exists. The **Blowfish** theme provides Tailwind CSS-based layouts, client-side search (Fuse.js), syntax highlighting, and responsive design. Content is authored in Markdown with TOML front matter. Deployment is fully automated via GitOps — pushes to `main` trigger a Hugo build on Cloudflare Pages and/or GitHub Pages. The CI pipeline validates builds and checks for broken links using `lychee`.

## Core Framework Stack

- **SSG**: Hugo v0.163.3 (extended) — Go-based static site generator
- **Theme**: Blowfish v2.103.0 — Tailwind CSS, Chart.js, Fuse.js, Mermaid, KaTeX, TypeIt
- **Hosting**: Cloudflare Pages (primary) + GitHub Pages (secondary)
- **CI/CD**: GitHub Actions (build check + lychee link checker, Pages deploy)

## UI & Interaction Libraries

The Blowfish theme bundles several client-side libraries (all included via the theme, not project-level dependencies):

- **Tailwind CSS** — Utility-first styling framework
- **Fuse.js** — Client-side fuzzy search
- **Chart.js** — Data visualisation
- **Mermaid** — Diagrams from text
- **KaTeX** — LaTeX math rendering
- **TypeIt** — Typing animations
- **lite-youtube-embed** — Lightweight YouTube embeds

## Development Tools Overview

- **Hugo CLI** — `hugo server -D` for local dev, `hugo --gc --minify` for production builds
- **Git submodules** — Theme managed as a submodule pointing to `nunocoracao/blowfish`
- **GitHub Actions** — CI build check and deploy automation
- **lychee** — Link checker run in CI
